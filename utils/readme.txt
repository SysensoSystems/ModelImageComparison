Model Image Comparison Tool
==========================================================================
Helps to compare two Simulink models and finds the difference 
between the models using image comparison approach.

Usage:
- Add the ModelImageComparison folder or ModelImageComparison/utils folder to the MATLAB path.
- Use "micTool" command to launch the tool.
- Use "Help" button to launch the detailed help document.

Refer "sample_models" folder for quick testing of this toolbox.

Syntax:
* >> micTool
* >> micTool('<ModelName or SystemPath>')
* >> micTool('<ModelName or SystemPath>','<ModelName or SystemPath>')
